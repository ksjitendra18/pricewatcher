# About Project

Price watcher is a full stack application that can use be used to track prices of products on ecommerce sites (currently have amazon and flipkart).

I achieve the purpose through web scraping. 

## Tools Used

Next.js
PlanetScale
Prisma
TailwindCSS
Cheerio
Axios
